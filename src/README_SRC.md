# src package

This folder contains modular Python files for the revised time-aware heart disease benchmark.

Main additions in this revised source package:

- repeated stratified lightweight holdout analysis
- budget compliance analysis
- Wilcoxon-Holm and Friedman statistical tests
- target sensitivity analysis for binary, ordinal 3-class, and original 5-class targets
- revised workflow figure generation

Use `run_all.py` only as a lightweight example. Heavy AutoML/DL experiments should be run step by step according to the manuscript protocol and available hardware.
